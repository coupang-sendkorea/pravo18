# Сайт юридической компании "Правовой Защитник"

Профессиональный сайт юридической компании с адаптивным дизайном.

## Функционал
- Адаптивный дизайн для всех устройств
- Мобильное меню
- Форма обратной связи
- Секции услуг, о компании, контакты
- Плавная прокрутка

## Технологии
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (Vanilla)
- Font Awesome иконки
- Google Fonts

## Как использовать
1. Замените контактные данные в файле `index.html`
2. Настройте цвета в `style.css`
3. Загрузите на хостинг или используйте GitHub Pages
